Detailed Description

template<typename... Ts>
struct chip::Variant< Ts >

Represents a type-safe union.

An instance of Variant at any given time either holds a value of one of its alternative types, or no value. Each type must define a unique value of a static field named VariantId.

Example: struct Type1 { static constexpr const std::size_t VariantId = 1; };

struct Type2 { static constexpr const std::size_t VariantId = 2; };

Variant<Type1, Type2> v; v.Set<Type1>(); // v contains Type1 Type1 o1 = v.Get<Type1>();
