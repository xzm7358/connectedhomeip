template<typename F >
CHIP_ERROR chip::SerializableU64SetBase::Serialize ( F  callback)
inline

Serialize the sparse array by calling a callback with a ByteSpan to serialize.

We ensure that this ByteSpan is architecture-agnostic, so it can be deserialized anywhere later.

Only the values till mNextAvailable index are encoded. The empty indexes between 0, and mNextAvailable, are also encoded.

Parameters
[in]callbackthe serialization callback to call.
